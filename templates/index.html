About This Demo

This demo allows you to use two different OCR engines for Khmer text recognition:

  • TrOCR - Transformer-based Optical Character Recognition model that works well with Khmer text.
  • Tesseract - Open-source OCR engine with Khmer language support.

For best results:

  • Use clear images with good lighting
  • Ensure text is well-focused and not blurry
  • Avoid excessive image noise or background patterns
